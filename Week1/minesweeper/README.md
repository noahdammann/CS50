## Overview

  - An AI that can play minesweeper

## Prerequisites

  - Python 3

## Installation

1. Clone the repo
  &nbsp;
   ``` git clone https://github.com/noahdammann/CS50 -b CS50AI ```
3. Move into minesweeper ``` cd CS50/Week1/minesweeper ```
4. Download requirements ``` pip3 install -r requirements.txt ```

## Usage

```
python runner.py
```
