## Overview

  - An AI that can play minesweeper

## Prerequisites

  - Python 3

## Installation

1. Clone the repo
   ```sh
    git clone https://github.com/noahdammann/CS50 -b CS50AI
   ```
2. Move into minesweeper
  ```sh
    cd CS50/Week1/minesweeper
   ```
3. Download requirements
   ```sh
     pip3 install -r requirements.txt
   ```

## Usage

```
python runner.py
```
